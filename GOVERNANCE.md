# RedRock Governance / 红岩治理公约（CN/EN）

## 1. Bilingual Rule / 双语规则
- All user-facing docs MUST be bilingual (CN first, EN second), or CN/EN side-by-side.
- 所有面向外部读者的材料必须双语（先中文后英文，或中英对照）。

## 2. Standard ID / 标准编号
- Format: RR-<DOMAIN>-<NAME>-<VERSION>
- Example: RR-NSE-EC-1.0
- 格式：RR-<领域>-<名称>-<版本>；示例：RR-NSE-EC-1.0

## 3. Versioning / 版本规则
- Use SemVer: MAJOR.MINOR.PATCH
- Release notes required for MINOR/MAJOR
- 使用语义化版本；MINOR/MAJOR 必须有发布说明

## 4. Release Process / 发布流程
1) Draft → Review → Release
2) Update registry entry
3) Update README hub links
4) Tag release if needed
