# RedRock Constitution / 红岩总宪章仓库

## Bilingual Governance Notice / 双语治理声明
本仓库作为红岩体系的“总入口与标准注册表”。所有对外材料默认双语（中文优先，英文紧随）。  
This repository is the hub of the RedRock system: registry + navigation. All public-facing docs are bilingual (CN first, EN follows).

## Architecture Overview / 架构总览
- Constitution Hub（本仓库）：标准编号、注册表、路线图、治理规则
- Domain Repos（各领域仓库）：标准草案、参考实现、演示与实验

## Repositories Map / 仓库导航

- AASP-Core：算法与代谢相关核心 / Algorithm & Metabolism Core
- ISAS-Core：工业自治标准与核心 / Industrial Autonomy Standard Core
- DOIP-Segments-Specification：DOIP 分段与模型规范 / DOIP Segment Spec
- MCP-Legal-China：中国法律语义与 MCP 工具链 / China Legal + MCP Tooling
- Kinetic-Robotics-FDO-Sovereignty：机器人/运动控制主权化探索 / Robotics Sovereignty
- China-Precision-Mold-FDO-Standard：精密模具领域标准化试点 / Precision Mold Standard
- aro-audit：高权限 AI 行为的证据层审计画像 / Evidence-layer audit profile for high-authority AI actions
- safety-valve-spec：动作边界安全阀与一致性规范 / Action-boundary safety valve and conformance spec
