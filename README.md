# RedRock Constitution

## Bilingual Governance Notice
**中文**：RedRock 标准体系要求所有标准文档强制采用中文与英文双语镜像发布，英文必须为完整翻译版本，不得以摘要替代。  
**English**: The RedRock standards system mandates full CN/EN mirrored publication for all standards documents. English content must be a complete translation and cannot be replaced by a summary.

## Architecture Overview / 架构总览
- **RedRock 是标准制定与主权架构体系**：作为跨仓库治理根节点，负责标准命名、版本演进与发布一致性约束。  
  **RedRock is a standards and sovereignty architecture system**: it serves as the cross-repository governance root for naming, version evolution, and release consistency.
- **下属标准族谱**：统一维护标准编号、状态、入口链接与双语镜像约束。  
  **Standards lineage under governance**: it centrally maintains standard IDs, status, entry links, and bilingual mirroring constraints.
- **产品实验入口**：通过标准入口映射到各仓库实验与产品界面，形成“标准—实现—验证”闭环。  
  **Product and experiment entrypoints**: it maps standards to implementation and product interfaces across repositories for a standards-implementation-validation loop.
