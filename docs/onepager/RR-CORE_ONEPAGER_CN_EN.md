## 1) What it is / 它是什么

**中文**  
**RedRock Constitution (RR-CORE)** 是红岩体系的总入口与标准注册表。本仓库维护标准编号、各域仓库导航、标准注册表与治理公约，不直接实现业务规范。

**EN**  
**RedRock Constitution (RR-CORE)** is the hub and standards registry of the RedRock system. This repository maintains standard IDs, repository index, standards registry, and governance rules; it does not implement domain specifications directly.

---

## 2) Why it matters / 为什么重要

**中文**  
- 统一入口：所有领域仓库与一页纸从本 Hub 可发现。  
- 可追溯：每个标准域有唯一 ID（如 RR-AASP、RR-pFDO），与实现仓库、一页纸一一对应。  
- 双语一致：对外材料默认中英同页（中文优先，英文紧随）。

**EN**  
- Single entry: All domain repos and onepagers are discoverable from this hub.  
- Traceability: Each standard domain has a unique ID (e.g. RR-AASP, RR-pFDO) mapped to its implementation repo and onepager.  
- Bilingual: Public docs are CN/EN on the same page (Chinese first, English follows).
