# Standards Registry / 标准注册登记册

## Registry Statement / 登记声明
**中文**：本登记册为 RedRock 标准体系的唯一注册入口，用于定义标准编号、标准状态、版本基线与规范链接。除非在本登记册完成注册，任何标准不得作为正式标准对外发布。  
**English**: This registry is the sole registration entrypoint of the RedRock standards system for standard IDs, status, version baselines, and normative links. No standard may be released as formal unless it is registered here.

## Identifier System / 编号体系
- **中文**：统一编号格式为 `RR-<DOMAIN>-<NAME>-<MAJOR>.<MINOR>`。  
  例如：`RR-NSE-EC-1.0`、`RR-DOIP-SEG-2.0`。  
- **English**: The unified identifier format is `RR-<DOMAIN>-<NAME>-<MAJOR>.<MINOR>`.  
  Examples: `RR-NSE-EC-1.0`, `RR-DOIP-SEG-2.0`.

## Status Definition / 状态定义
- **Draft**：草案阶段，允许结构调整，不具备对外稳定承诺。  
- **Release**：发布阶段，具备稳定接口与语义承诺。  
- **Experimental**：实验阶段，用于验证机制，可能发生非兼容调整。  

- **Draft**: Proposal stage; structural changes are allowed and no external stability commitment is guaranteed.  
- **Release**: Published stage with stable interface and semantic commitments.  
- **Experimental**: Validation stage; non-backward-compatible changes may occur.

## Conformance Note / 一致性说明
**中文**：所有登记标准必须满足 CN/EN 全量镜像、编号唯一、版本可追踪、链接可达四项最小一致性要求。  
**English**: All registered standards shall meet four minimum conformance requirements: full CN/EN mirroring, unique identifier, version traceability, and reachable canonical links.
