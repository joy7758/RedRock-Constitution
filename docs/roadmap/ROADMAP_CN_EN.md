# Roadmap / 路线图（CN/EN）

## Phase 1 (0-6 months): Standard Consolidation / 标准收敛
- Consolidate glossary and IDs; unify schema conventions.
- 统一术语表与编号体系；固化协议字段与校验。

## Phase 2 (6-18 months): Experimental Governance / 实验治理
- Build reference implementations for each standard.
- 为每个标准提供可复现参考实现与演示入口。

## Phase 3 (18-36 months): Sovereignty Infrastructure / 主权基础设施
- Interop, compliance mapping, multi-party verification.
- 跨系统互操作、合规映射、第三方验证与审计机制。
